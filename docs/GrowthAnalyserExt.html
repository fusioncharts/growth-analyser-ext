An extension for FusionTime XT charts. Helps to analyse growth of data based on posisitions specific formulae and custom values. Growth analyser supports pre analysis of data which can be provided in chart configuration
Example
{
  datasource: {
    extensions: {
      'growth-analyser': {
        'growthOver': 'firstIndex'
       }
    }
  }
}

Available 'growthOver' options are 'firstIndex', 'prevIndex', 'Minimum', 'Maximum',
'Average', 'Median' & 'Standard Deviation'.
